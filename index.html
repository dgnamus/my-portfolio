DevOps

I built and automated a CI/CD pipeline for a web application using GitHub Actions. The pipeline handles code integration steps including dependency installation, linting, testing, and building the application into production-ready artifacts. I implemented artifact management within the pipeline and automated the deployment process to a virtual machine environment, ensuring that every change pushed to the repository can be reliably built and deployed. As part of the project, I worked with Docker to containerize the application and began structuring the deployment in a more reproducible way. I also configured a reverse proxy using Nginx and set up HTTPS to simulate a production-like environment. This project helped me understand the full lifecycle of an application—from code commit to deployment—as well as the fundamentals of automation, environment consistency, and reliable delivery.
